### 383 赎金信

```python 
class Solution:
    def canConstruct(self, ransomNote: str, magazine: str) -> bool:
        from collections import Counter
        a = Counter(ransomNote)
        b = Counter(magazine)
        for i,j in a.items():
            if j > b[i]:
                return False
        return True
```
